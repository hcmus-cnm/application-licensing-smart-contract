# Application licensing smart contract

## Getting started

### Requirements

- Truffle

  ```
  npm install -g truffle
  ```

- Ganache
- Solidity Compiler
  ```
  npm install -g solc
  ```

### Installation

```
cd frontend
npm install
```

### Runs the app in the development mode

- Run web app
  ```
  cd frontend
  npm start
  ```
